## How to retrieve systematic weight index information

First retrieve the dictionary of the systematic weight indices

```bash
setupATLAS
lsetup rucio
rucio download --nrandom=1 mc16_13TeV.410472.PhPy8EG_A14_ttbar_hdamp258p75_dil.merge.AOD.e6348_e5984_s3126_r9364_r9315
asetup AthDerivation,21.2.99.0
checkMetaSG.py <filename> | grep HepMCWeightNames
```

Then process this dictionary in python

```python
d = # paste the dict you extracted with the checkMetaSG.py script
for name, index in sorted(d.items(), key=lambda item: item[1]):
    print("{0}: {1}".format(index, name))
```
